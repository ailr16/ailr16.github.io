TI-NSPIRE CALCULATOR PROGRAMMING BLOG

ABOUT THE PROJECT

This is a blog I created with resources for
TI-Nspire CX calculators programming, using its
native language TI-BASIC. There're some examples
from basic to advanced complexity (as evaluating
Pythagorean theorem or getting the Trigonometric
series for discontinuous functions).
