INMOOV HAND & NEUROSKY MINDWAVE

ABOUT THE PROJECT

The project consists in a 3d printed robotic hand based
on InMoov project which is controlled with attentions levels
provided by a Neurosky Mindwave Mobile Headset.

Also, I developed an Arduino library for simplifying headset
implementation in projects.
