ABOUT THE PROJECT

Since three years ago I involved in Embedded Development using
a Raspberry Pi 4 board and Raspberry Pi OS.
Due the project was the first version of developing a useful device,
all the deployment was manual with a little implementation of devops
using GitHub, but that is not the better way to deploy many devices.

As part of further research, I discovered the Yocto project and other
tools like Buildroot to build custom Embedded Linux images.
Until these days I know how to set the environment for cross compilation
with Docker containers and Virtual Machines and a little familiarity with
building process but I still experimenting with these amazing tools
that opens up a world of possibilities in Embedded devices.
